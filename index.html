Reporting Notes
  • Excluded channels: "Other" and "Events" channels are excluded from all views in this dashboard.
  • Data discrepancies: Total pipeline data may differ slightly from the CY26 Acquisition Model due to how we pull the Salesforce data. In order to get a complete view, we rely on Lead ID, Opportunity ID, and other fields to match up every record, so there are small discrepancies on some channels — but they are marginal.
  • First touch attribution: All records (no Inbound filter) using Original Marketing Channel for MQLs. S2 first touch uses Original Marketing Channel sourced from Pipeline Data with Campaign via Opportunity ID.
  • Last touch attribution: Source Mapped = Inbound, using Channel Mapped for both MQL and S2.
  • Landing Pages tab: Traffic data is sourced from GA4 (Landing Page dimension). MQL and S2 attribution is matched by joining GA4 landing page paths with HubSpot "HS First Page Seen" and "HS Last Page Seen" fields from Pipeline Data. The URL join strips the domain and query parameters from HubSpot URLs to match GA4's path-only format (e.g. /blog/revenue-streams). Match rate is ~99%. Non-roller.software URLs (e.g. meetings.hubspot.com) are excluded. The blue bar in the Traffic column shows relative traffic volume compared to the highest-traffic page on the current page of results.
